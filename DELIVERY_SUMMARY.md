```jsx

```

### Change Colors
File: `src/index.css` or `tailwind.config.js`
```css
#ffd700  →  your-color
```

### Add Button Sound
File: `src/components/MainMenu.jsx`
```jsx
// Add audio play on click
```

### Change Background Image

```jsx

```

### Adjust Animation Speed
File: `tailwind.config.js`
```javascript
"0.6s" → "0.3s" (faster) or "1s" (slower)
```
